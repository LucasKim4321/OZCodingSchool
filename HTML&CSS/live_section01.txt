## HTML
-----------

HTML은 마크업 언어(프로그래밍 언어가 아님)

HTMl(HyperText Markup Language)

HyperText : 하이퍼링크를 통해 어떤 문서에서 다른 문서로 접근할 수 있는 텍스트
Markup : (콘텐츠를)표시하다. 보이는 영역(클라이언트)의 구조를 세우는 작업
Language : 언어

<>로 감싸진 모든 것을 태그

태그가 특정 역할을 하면서 콘텐츠를 포함할 수 있는 구조 전체를 '요소'라고 한다!

요소의 하위 요소를 자식요소라고 부른다
부모요소(parent)
자식요소(child)

HTML문서는 계층 구조를 이루고 있다.

각 요소는 특정 기능을 수행하기 위해 속성(attributes)을 가질 수 있다.
<a href="https://www.naver.com">네이버</a>

요소를 선택할 대는 선택자(selector)를 사용한다.
종류가 많음

id 선택자
문서 전체에서 고유한 선택자를 지정.
css에서 #id로 선택

class 선택자
모든 요소가 사용가능한 고유하지 않은 선택자를 지정.
요소는 여러개의 클래스를 가질 수 있음.
css에서 .class로 선택

커맨드 시프트 l : 같은 요소 전체 선택
옵션 클릭 : 커서 생성. 여러개 동시 입력,수정,삭제 가능
