🐾바다의 방랑자, 돌고래 '루카'🐾

이름: 루카(Luca)

종: 대서양 큰돌고래(bottlenose Dolphin)

성별: 수컷

출생지: 제주 근해

현재 거주지: 남해 바다

🌊루카의 바다 일기🌊

안녕하세요! 저는 자유롭게 바다를 헤엄치는 돌고래 루카입니다.
푸른 바다를 가르며 친구들과 노는 게 제일 좋아요!

매일 해변 근처에서 점프 연습을 하고, 가끔 사람들에게 호기심이 생기면 가까이 다가가기도 해요.
특히 배가 지나갈 때 파도를 타는 건 정말 짜릿하답니다! 🐬💦

🐾 루카의 특징

✅ 속도광 돌고래 – 빠르게 헤엄치며 바다를 질주하는 걸 좋아해요!
✅ 소통왕 – 다양한 소리와 몸짓으로 친구들과 대화를 나눠요.
✅ 장난꾸러기 – 물장구를 치거나 물고기를 쫓아다니며 하루를 보냅니다.

🐬 언젠가 바다에서 저를 만나면 반갑게 손 흔들어 주세요! 🌊
